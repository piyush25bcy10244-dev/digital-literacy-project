## Project Overview

This project was about understanding how to use digital tools in a safe and efficient way. It included making an infographic about digital
literacy, creating online profiles, exploring platforms like HackerRank and Google Forms, writing professional emails, and learning about cybercrime.  

While doing this project, i was asked to try several digital tools. Many of them were new to me but i learnt a lot from them. It helpedd me understand things that will be very useful in my day to day college life.

## Task Summary

### Task 1 – Infographic
I created a digital literacy infographic using Canva and learnt how to present information in a simple and visual way.

### Task 2 – Digital Portfolio
I set up profiles like GitHub and learned about maintaining a professional online presence.

### Task 3 – Platforms
Used HackerRank to practice a basic "Hello World" problem and created a Google Form quiz and shared with batchmates for digital literacy awareness.

### Task 4 – Email Etiquette
I wrote professional emails as a student and created a checklist for responsible social media usage.

### Task 5 – Cybercrime
In this task I studied a UPI fraud case and created a prevention checklist to spread awareness.
